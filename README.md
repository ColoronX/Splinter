# Splinter

A minimal, fast, and lightweight video cropping application. 

Splinter provides a simple and precise way to crop videos entirely in your browser. By utilizing a freeform selection tool, you can simply drag a crop box over your video to get the exact framing you need without relying on external servers.

## Features

* Freeform drag-and-drop selection
* Precise, visual crop adjustments
* Client-side processing via ffmpeg
* Strict, minimal user interface
Hosted on: https://coloronx.github.io/Splinter/
